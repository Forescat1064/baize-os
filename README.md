# Baize OS

Baize OS is a self-contained, local-first personal memory and execution web app.

## Open

Open `index.html` in a modern browser. No installation or server is required.

## Data

All notes, tags, priorities, selections, and theme preferences are stored in the
browser's local storage. The app does not send data to a server.

## Features

- Seven workspaces: Profile, Long-term Goals, Current Plan, Daily Log, Projects,
  Ideas, and Prompt Library
- Create, edit, delete, and save notes
- Full-text search across all workspaces
- Note tags and tag filtering
- Select notes and export a Markdown or text AI context prompt
- Light and dark themes
- Responsive sidebar and mobile layout
- Keyboard shortcuts: `Ctrl/Cmd + K` for search and `Ctrl/Cmd + S` to save
